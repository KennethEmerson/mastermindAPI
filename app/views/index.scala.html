Mastermind API

This basic API allows you to play a game of Mastermind
the various GET requests are the following:
Page URL Description
ask to start a new game /mastermind/newgame/x/y the x represents the number of pins, y the number of colors.
ask to evaluate a guess /mastermind/evaluation/? the ? represents the guessed color pins as integers divided by a . example : 1.2.3.4
ask to return the solution /mastermind/solution returns the solution
